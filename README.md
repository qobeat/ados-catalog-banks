# ados-catalog-banks

Generated, append-only JSONL data banks for the ADOS catalog.

- `banks/` append-only ledgers + mutable `restore_point.json`
- `bank-descriptors/` bank descriptors
- `reference-model-banks/` ADOS archetype/domain reference
- `semantic-index/` retrieval store
- `logs/` structured logs
- `reports/` JSON reports

Do not place application source code here. Never commit secrets.
